1、安装中文输入法
方法一(fcit-google pinyin输入法)
修改更新源:

vim /etc/apt/sources.list 
#阿里源
deb http://mirrors.aliyun.com/kali kali-rolling main non-free contrib
更新数据和软件包
apt-get update
apt-get upgrade

安装输入法框架fcitx:

 apt-get install fcitx
 #安装google拼音输入法
 apt-get install fcitx-googlepinyin
 #重启系统
 reboot

打开Fcitx Configuration 把Gooogle Pinyin输入法添加到“Input Method”首行

卸载方法:

apt-get remove fcitx
apt-get remove fcitx-googlepinyin

方法二(ibus-pinyin输入法) 同样要修改更新源，和上面一样好吧。 安装输入法框架ibus:

apt-get install ibus ibus-pinyin

im-config    #选择ibus(截图工具scrot闪退就不发图片了-_-)

#重启
reboot

设置输入法:

ibus-setup
#窗口选择
Input Method -> Add -> Chinese-Pinyin

打开设置-Region&Language-Inpur Sources 添加Chinese（Pinyin）

2、kali linux额外扩展
hardoopad (linux最好用的markdown编辑器)安装教程:
官网下载:http://pad.haroopress.com/user.html
deb包安装:

dpkg -i haroopad-v0.13.1-x64.deb

依赖包错误:

/usr/share/haroopad/haroopad: error while loading shared libraries: libgconf-2.so.4: cannot open shared object file: No such file or directory

apt install libgconf2-4 错误:

Reading package lists... Done
Building dependency tree       
Reading state information... Done
Package libgconf2-4 is not available, but is referred to by another package.
This may mean that the package is missing, has been obsoleted, or
is only available from another source
However the following packages replace it:
 gconf-gsettings-backend

E: Package 'libgconf2-4' has no installation candidate

解决依赖包错误:

apt install gconf-gsettings-backend

bundle安装:

chmod +x xxx.bundle
./xxx.bundle

截图工具scrot

#安装
apt-get install scrot
#截取整个屏幕,保存到当前目录
scrot
#保存到指定目录
scort /root/Desktop/2020.png
#截取指定屏幕
scrot -s /root/Desktop/2020.png

图片、pdf打开工具xdg-open

#使用
xdg-open xxx.png

视频播放工具mplayer

#安装
apt-get install mplayer
#使用
mplayer xxx.avi
