linux-kali烧写4412开发板（最小linux文件系统）

January 25, 2020 4:29 PM

工具： 串口助手

apt-get install minicom

安装adb：

apt-get install adb  或    
apt-get install android-tools-adb

安装fastboot：

apt-get install android-tools-fastboot

文件准备： Uboot 镜像 内核镜像“zImage” 根文件系统“ramdisk-uboot.img”和“system.img”

串口和OTG分别连接电脑。

判断adb是否连上设备

adb devices

启动4412进入root模式 打开minicom设置串口属性

minicom -s

#查看分区
fdisk -c 0

#分区格式化命令，超级终端中输入命令：

fatformat mmc 0:1
ext3format mmc 0:2
ext3format mmc 0:3
ext3format mmc 0:4  

#输入连接命令，超级终端中输
fastboot

cd到文件目录下启动fastboot：

#烧写uboot
fastboot flash bootloader uboot镜像名字

#烧写zImage内核：
fastboot flash kernel zImage

#输入烧写根目录根目录文件系统ramdisk:
fastboot flash ramdisk ramdisk-uboot.img

#输入烧写system 文件系统命令:
fastboot flash system system.img

#输入擦除命令：
fastboot -w

#输入重启开发板命令：
fastboot reboot

附加（磁盘）知识： 硬盘分区查看

fdisk -l

硬盘修复

ntfsfix /dev/sda3

硬盘格式化

mkfs -t ntfs ./sda3
